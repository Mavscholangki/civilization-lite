# civ6-local

本项目是由三人团队（均来自于黄渡理工大学）开发的策略类游戏原型，旨在致敬并复刻经典大作《文明VI》的核心玩法。游戏定位为一款集回合制策略与城市模拟建设于一体的“微缩版”4X游戏（探索、扩张、开发、征服）。玩家将在随机生成的六边形地图上扮演一位文明领袖，体验从建立第一座城市、发展基础科技，到组建现代化军队、争夺世界霸权的完整文明演进历程。

在游戏开局阶段，玩家需从三大特色文明中进行抉择：擅长工业制造与生产力爆发的德国、拥有广阔领土与深厚文化底蕴的俄罗斯，以及精通科技革新与“尤里卡”时刻的中国。游戏地图采用程序化生成技术，包含了平原、森林、山脉、河流等丰富地形及动态的战争迷雾系统，确保了每一局游戏都拥有全新的探索体验与战略布局挑战。

城市建设与内政运营是国力强盛的基石。游戏内的城市拥有人口、生产力、城墙耐久及生命值等核心属性。玩家需要在有限的地块上规划建设各类功能区域，例如产出科技值的学院、提供生产力的工业区、增加文化产出的剧院广场以及作为科技胜利关键的宇航中心。合理的区域规划与资源管理，是维持庞大帝国运转并支持前线战事的关键。
为了模拟历史的进程，项目设计了严谨的双轨制成长体系。其中，“科技树”决定了硬实力的上限，负责解锁从勇士到喷气式战机的各时代兵种；“文化树”则代表软实力的积累，用于解锁各类强力的政策卡。特色的尤里卡系统也被完整保留，玩家可以通过特定的游戏行为（如击败野蛮人、建造特定设施）来触发科研灵感，大幅加速发展进度，这极大地丰富了玩家的决策维度。

在军事对抗与胜利机制方面，游戏构建了包含近战、远程、攻城及空军在内的四条完整兵种升级路线，并引入了城墙攻防机制。与之对抗的是基于 A\* 寻路算法与决策模型构建的 AI 对手，它们具备自动发展、评估战力及发动战争的能力。玩家最终的目标是通过武力彻底摧毁所有敌对文明的首都以取得征服胜利，或集举国之力建成宇航中心并完成登月项目，从而达成科技胜利。
